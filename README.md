Proyek ini adalah aplikasi CRUD Data Mahasiswa sederhana yang dibuat menggunakan PHP Native dan MySQL. Aplikasi ini berfungsi untuk mengelola data mahasiswa, seperti menambah, menampilkan, mengubah, dan menghapus data.

Pada halaman utama, data mahasiswa ditampilkan dalam bentuk tabel yang berisi Nomor, Nama, NIM, serta menu Aksi berupa tombol Edit dan Hapus. Pengguna juga dapat menambahkan data baru melalui menu Tambah Data yang berada di bagian atas halaman.

Aplikasi ini dibuat dengan tampilan sederhana menggunakan HTML dan CSS, dan cocok digunakan sebagai latihan dasar untuk memahami konsep CRUD di PHP dan database MySQL.
